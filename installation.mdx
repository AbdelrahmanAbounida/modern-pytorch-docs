## Quick Install

Choose your preferred installation method and compute platform.

<Tabs>
  <Tab title="pip">
    ### CPU Only
    ```bash
    pip install torch torchvision torchaudio
    ```

    ### CUDA 11.8
    ```bash
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
    ```

    ### CUDA 12.1
    ```bash
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
    ```

    ### MPS (Apple Silicon)
    ```bash
    pip install torch torchvision torchaudio
    ```
    <Note>MPS backend is included by default on macOS builds.</Note>
  </Tab>

  <Tab title="conda">
    ### CPU Only
    ```bash
    conda install pytorch torchvision torchaudio cpuonly -c pytorch
    ```

    ### CUDA 11.8
    ```bash
    conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia
    ```

    ### CUDA 12.1
    ```bash
    conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia
    ```

    ### MPS (Apple Silicon)
    ```bash
    conda install pytorch torchvision torchaudio -c pytorch
    ```
  </Tab>

  <Tab title="From Source">
    ### Prerequisites
    ```bash
    conda install cmake ninja
    pip install -r requirements.txt
    ```

    ### Clone and Build
    ```bash
    git clone --recursive https://github.com/pytorch/pytorch
    cd pytorch
    git submodule sync
    git submodule update --init --recursive
    ```

    ### CPU Only
    ```bash
    export USE_CUDA=0
    python setup.py develop
    ```

    ### With CUDA
    ```bash
    export USE_CUDA=1
    python setup.py develop
    ```

    <Warning>Building from source requires significant time and system resources.</Warning>
  </Tab>
</Tabs>

## Verify Installation

Run the following to confirm PyTorch is installed correctly:

```python
import torch

# Check version
print(f"PyTorch version: {torch.__version__}")

# Check available devices
print(f"CUDA available: {torch.cuda.is_available()}")
print(f"MPS available: {torch.backends.mps.is_available()}")

# Test tensor creation
x = torch.rand(3, 3)
print(f"Tensor:\n{x}")
```

### Verify CUDA

```python
if torch.cuda.is_available():
    print(f"CUDA version: {torch.version.cuda}")
    print(f"GPU: {torch.cuda.get_device_name(0)}")
    
    # Test GPU tensor
    x = torch.rand(3, 3, device="cuda")
    print(f"GPU Tensor:\n{x}")
```

### Verify MPS (Apple Silicon)

```python
if torch.backends.mps.is_available():
    device = torch.device("mps")
    x = torch.rand(3, 3, device=device)
    print(f"MPS Tensor:\n{x}")
```
