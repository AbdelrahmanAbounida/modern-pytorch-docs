## What's Included

This documentation covers **all 2000+ PyTorch APIs** organized into:

<AccordionGroup>
  <Accordion title="Core Tensor Operations" icon="cube">
    - Tensor creation (zeros, ones, arange, linspace, etc.)
    - Tensor manipulation (reshape, transpose, permute, etc.)
    - Indexing and slicing operations
    - Tensor attributes and views
  </Accordion>

<Accordion title="Mathematical Operations" icon="calculator">
    - Pointwise operations (add, mul, sin, exp, etc.)
    - Reduction operations (sum, mean, max, etc.)
    - Comparison and logical operations
    - Bitwise operations
  </Accordion>

  <Accordion title="Neural Network Modules" icon="brain">
    - Convolution layers (Conv1d, Conv2d, Conv3d)
    - Pooling layers (MaxPool, AvgPool, AdaptivePool)
    - Activation functions (ReLU, GELU, Sigmoid, etc.)
    - Normalization (BatchNorm, LayerNorm, GroupNorm)
    - Recurrent layers (RNN, LSTM, GRU)
    - Transformer layers and attention mechanisms
  </Accordion>

  <Accordion title="Training & Optimization" icon="chart-line">
    - Loss functions (MSE, CrossEntropy, etc.)
    - Optimizers (SGD, Adam, AdamW, etc.)
    - Learning rate schedulers
    - Automatic differentiation (autograd)
  </Accordion>

  <Accordion title="Advanced Features" icon="wand-magic-sparkles">
    - Distributed training (DDP, FSDP)
    - JIT compilation (TorchScript)
    - PyTorch 2.0 compiler (torch.compile)
    - Quantization and pruning
    - Model export (ONNX, TorchScript)
    - Profiling and debugging
  </Accordion>

  <Accordion title="Specialized Operations" icon="sparkles">
    - Linear algebra (torch.linalg)
    - Signal processing (torch.fft)
    - Sparse tensors
    - Nested tensors
    - Special functions
  </Accordion>
</AccordionGroup>
